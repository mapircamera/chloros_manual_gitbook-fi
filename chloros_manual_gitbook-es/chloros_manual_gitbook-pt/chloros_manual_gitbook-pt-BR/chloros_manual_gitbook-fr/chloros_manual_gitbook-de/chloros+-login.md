## CLI-kirjautuminen

Ota CLI-käsittely käyttöön kirjautumalla sisään Chloros+ -tunnuksillasi.

**Syntaksi:**

```bash
chloros-cli login <email> <password>
```

**Esimerkki:**

```powershell
chloros-cli login user@example.com 'MyP@ssw0rd123'
```

{% hint style="warning" %}
**Erikoismerkit**: Käytä lainausmerkkejä salasanoissa, jotka sisältävät merkkejä, kuten `$`, `!` tai välilyöntejä.
{% endhint %}

### Suunnitelman päättyminen

GUI:n suunnitelman vanheneminen näyttää, milloin lisenssisi raukeaa. Toistuvien kuukausitilausten voimassaoloaika päättyy kuun lopussa. Vuositilauksilla se on vuosi tilauksen aloittamisesta. Lisenssitarkistus vaatii kuukausittaisen Internet-yhteyden vahvistamista varten 30 päivän lisäajan kanssa.

### Laitteen rajoitus

Kukin Chloros+ -paketti tarjoaa eri määrän rekisteröityjä laitteita. Jokainen laite, johon kirjaudut sisään Chloros+ -tilillä, lasketaan mukaan rekisteröityjen laitteiden määrään. Voit nimetä uudelleen ja poistaa laitteen MAPIR Cloud -tilisivullasi.
